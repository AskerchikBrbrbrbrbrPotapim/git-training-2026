git init - создание новой папки/репозитория
git clone - скачивает проект
git status - ну типо статус файлов какие добавлены и готовы к отправке какие ещё нет какие есть новые файлы
git add - отправление файлов на подготовку
git commit - сохраняет изменения но локально
git branch - создает новую ветку разработки
git checkout - перемещение между ветками
git push - отправляет добавленные файлы (которые были добавлены командой git add) уже на облако? гитхаб?
